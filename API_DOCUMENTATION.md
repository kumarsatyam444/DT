# Task Management API Documentation

## Base URL

http://localhost:3000/api


## Overview
A RESTful API for creating, reading, updating, and deleting tasks.


## Status Codes
| Code | Description |
|------|-------------|
| 200 | Success |
| 201 | Created |
| 400 | Bad Request |
| 404 | Not Found |

---

## Start

npm install
npm start

Server runs at `http://localhost:3000`
